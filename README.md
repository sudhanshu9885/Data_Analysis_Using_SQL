
# Pizza Sales Data Analysis Using SQL  
**Analyze sales performance, customer behavior, and menu insights using SQL and Excel**  

**Deliver actionable insights into pizza sales, trends, and product profitability for data-driven business decisions.**  

---

## Tools & Technologies
- **Data Preparation:** Microsoft Excel  
- **Database & Analysis:** SQL Server, SQL  

---

## Key Features

- **Data Preparation & Ingestion:**  
  Downloaded the Pizza Sales dataset from Kaggle and performed initial cleaning and preprocessing in Excel. Handled inconsistencies, validated data quality, and imported the cleaned dataset into SQL Server for structured analysis.

- **SQL-Based Data Analysis:**  
  Executed SQL queries to calculate key business metrics such as total revenue, total orders, total pizzas sold, average order value, and average pizzas per order.

- **Time-Based Trend Analysis:**  
  Analyzed hourly and weekly sales trends by extracting time and date components. Identified peak ordering hours, daily demand patterns, and weekly order trends to understand customer purchasing behavior.

- **Category & Size Performance Analysis:**  
  Evaluated sales contribution by pizza category and size. Calculated percentage distribution of revenue and order volume to identify top-performing and underperforming segments for menu optimization.

- **Product Performance Insights:**  
  Used CTEs and aggregations to determine top and bottom-performing pizzas based on revenue, quantity sold, and number of orders. Highlighted best-selling and low-performing products to guide business strategy.

---

## Business Impact
Delivered actionable, data-driven insights into sales performance, customer demand, and product profitability. Supported improved decision-making around pricing, inventory planning, and menu strategy.
